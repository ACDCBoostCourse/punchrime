friend에 대해 설명하시오

friend : 하나의 클래스에서 다른 클래스의 내부 데이터에 접근해야하는 경우 권한을 주고 싶을 때 사용

*함수나 클래스의 선언 앞에서 선언 가능
 friend를 선언한 함수나 클래스의경우 접근 제어 지시자의 영향 받지 않음

class PeopleA{
private : 
	string name;
	int height;
	friend class PeopleB;  //PeopleB에게 권한을 줌
public : 
	PeopleA(string name, int height) {
		this->name = name;
		this->height = height;
	}
};
